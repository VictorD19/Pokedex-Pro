## Pokedex Pro

Contem uma lista de 500 pokemon

Extructura de um pokemon:
{

    name:'',
    id:'',
    types:[],
    stast:[],
    img:'',
    abilitys:[]
}

### Faça um clone do repositorio

~~~
git clone https://github.com/VictorD19/Pokedex-Pro.git
~~~

### Inicie o npm
~~~
npm i
~~~

### Inicie o projecto
~~~
npm start
~~~
